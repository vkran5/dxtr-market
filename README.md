# DXTR Market

A cryptocurrency trading dashboard with authentication flow.

## Features

- **Authentication** - Login with email or phone number, OTP verification
- **Protected Routes** - Market page requires authentication via middleware
- **Crypto Market View** - Browse cryptocurrency listings with filters
- **Search & Filter** - Filter by category (All, Crypto, Favorites, DeFi, Meme, Hot)
- **Responsive Design** - Mobile-first, works on desktop and mobile

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4
- **Data Fetching**: SWR
- **Icons**: Lucide React

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm run start
```

Open [http://localhost:3000](http://localhost:3000) to view the app.
